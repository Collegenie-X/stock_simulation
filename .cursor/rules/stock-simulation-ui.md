# 주식 시뮬레이션 게임 UI 규칙

## UI 디자인 원칙

### 토스 스타일 적용
- 금액 표시: 큰 숫자를 강조, 수익률은 보조 정보로 작게
- 정보 위계: 중요한 정보(총 자산, 수익 금액)를 크게, 부가 정보(%, 날짜)는 작게
- 반올림: 모든 금액은 `Math.round()`로 정수 표시 (소수점 없음)

### 레이아웃 구조
- Sticky 헤더: 중요한 정보는 스크롤해도 항상 보이도록 상단 고정
- 컴팩트 디자인: 불필요한 여백 최소화, 정보 밀도 높게
- 3단 구조: 내 주식 → 관심 주식 → 전체 주식

### 인터랙션
- 하트 버튼: 관심 주식 등록/해제
- 탭 전환: 현재가/평가금 토글
- 일시정지: 타이머 일시정지 기능

## 금액 표시 규칙

```typescript
// ✅ 좋은 예
<div>{Math.round(totalValue).toLocaleString()}원</div>

// ❌ 나쁜 예
<div>{totalValue.toFixed(2)}원</div>
```

## 정보 위계

```
총 자산 (text-base, 보조)
10,234,567원 (text-xl, 주인공)
+12.3% (text-xs, 작은 뱃지)
```
