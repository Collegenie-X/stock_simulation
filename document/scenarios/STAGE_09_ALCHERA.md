## 🧠 멘탈 관리 가이드

```mermaid
flowchart TD
    Situation["극심한 손실 상황"] --> Emotion{"감정 상태?"}
    
    Emotion -->|"공포/패닉"| Bad["❌ 잘못된 대응<br/>• 던지기 (전량 손절)<br/>• 물타기 남용<br/>• 무작위 베팅"]
    
    Emotion -->|"침착/냉정"| Good["✅ 올바른 대응<br/>• 상황 분석<br/>• 에너지 확인<br/>• 계획적 행동"]
    
    Bad --> Result1["게임오버 위험 ↑"]
    Good --> Result2["생존 + 회복 기회"]
```

### 멘탈 관리 체크리스트

| 상황 | 잘못된 반응 | 올바른 반응 |
|------|-----------|-----------|
| 급락 시 | 패닉 → 전량 손절 | 침착 → 단계적 대응 |
| 연속 BAD | 분노 → 복수 베팅 | 냉정 → 휴식 (0%) |
| 에너지 위기 | 공포 → 물타기 남용 | 계산 → 에너지 보존 |
| 반등 시작 | 조급 → 올인 | 확인 → 분할 진입 |
