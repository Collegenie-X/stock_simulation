## 1. Executive Summary

**"파도를 타라"**는 AI/로봇주의 실제 지난 1년 데이터 위에서, 나와 성향이 비슷한 AI와 **같은 주식으로 동시에 대결**하며 **차트를 읽는 감각**과 **큰돈을 다루는 감각**을 체득하는 **주식 투자 훈련 앱**입니다.

### 핵심 한 줄

> **"5,000만원으로 엔비디아를 매수한 순간, 유사 AI는 분할 매수했고, 최고 AI는 이미 매도했다. 그 차이를 30초 안에 느끼는 것 — 이것이 훈련이다."**

### 이 앱이 훈련하는 4가지 감각

| # | 감각 | 훈련 방법 |
|:---:|:---|:---|
| **1** | **차트를 읽는 힘** | 패턴 → 시나리오 → 실전 시뮬레이션, 3단계로 엘리엇 파동의 흐름을 몸에 새김 |
| **2** | **돌발 이벤트 대처력** | 실제 데이터 기반 급등/급락 시나리오에서 30초 안에 판단하는 반복 훈련 |
| **3** | **AI와 같은 주식으로 대결** | 내가 산 바로 그 종목에서 유사 AI와 최고 AI가 뭘 했는지 즉시 비교 |
| **4** | **큰 액수의 돈 감각** | 500만원~5억원 단위 가상 자본으로 실전과 같은 금액 감각을 미리 체득 |

### 왜 다른가

기존 시뮬레이션 앱은 **가짜 데이터로 혼자 연습**합니다.

"파도를 타라"는 **AI/로봇주의 실제 지난 1년 주가 데이터** 위에서, **나와 비슷한 성향의 AI가 같은 종목을 동시에 거래**하고, 그 격차를 실시간으로 보여줍니다. 차트의 흐름을 감각적으로 느끼고, 큰 금액에서 오는 심리적 압박을 미리 경험하는 것 — 이것이 단순 모의투자와 "훈련"의 차이입니다.

## ★ 우리가 훈련하는 것 — 이 앱의 존재 이유

> **"주식 투자는 지식이 아니라 감각이다. 감각은 읽어서 배우는 것이 아니라, 반복해서 느끼는 것이다."**

### ★-1. 차트를 읽는 힘 — 시나리오 → 패턴 → 시뮬레이션

유튜브 강의 100편을 봐도 실전에서 차트를 보면 얼어붙습니다. **차트를 읽는 힘은 이론이 아니라 반복 훈련으로만 길러집니다.**

```
[3단계 차트 감각 훈련 시스템]

Step 1. 시나리오 학습 (5턴 × 30초)
  "AI 반도체 급등 시나리오"
  → 엔비디아가 실적 발표 후 +12% 급등
  → 이 차트를 보고 30초 안에 매수/매도/관망 결정
  → AI는 같은 상황에서 뭘 했는지 즉시 비교
  → "아, 최고 AI는 1턴에서 이미 70% 매수했구나"

Step 2. 패턴 연습 (15초 타이머)
  "3파 상승 패턴"
  → 엘리엇 파동의 1파→2파 조정 후, 3파 시작 지점을 포착
  → 15초 안에 판단 — 빠르게, 감각적으로
  → 연속 정답 시 콤보 보너스 → 패턴이 몸에 새겨짐

Step 3. 실전 시뮬레이션 (100턴)
  "실제 AI/로봇주 1년 데이터"
  → 하루 3번 의사결정 × 30초 타이머
  → 100턴 동안 차트의 상승, 하락, 횡보, 급등, 급락을 모두 경험
  → 끝나면 "차트가 보이기 시작한다"는 감각이 생김
```

**핵심**: 시나리오로 개념을 잡고 → 패턴으로 반사 신경을 만들고 → 시뮬레이션으로 실전 감각을 완성합니다. 이 3단계를 거치면 **차트를 "읽는" 것이 아니라 "느끼는" 수준**에 도달합니다.

### ★-2. 돌발 이벤트 대처 능력 — 급등/급락에서 얼어붙지 않는 훈련

실전에서 가장 큰 손실은 **돌발 상황에서 아무것도 못 하고 얼어붙을 때** 발생합니다.

```
[실제 데이터 기반 돌발 이벤트 시나리오]

시나리오: "바이오 임상 실패"
  Day 15: 셀트리온 -8.5% 급락 (뉴스: "3상 임상 실패")
  → 30초 타이머 시작
  → 손절? 관망? 물타기?
  → 유사 AI: "50% 즉시 손절" (이유: 손절선 -5% 초과)
  → 최고 AI: "전량 손절 후 반등 대기" (이유: 추가 하락 가능성)
  → 나: 아무것도 못 함 (시간 초과 → 자동 관망)
  → 피드백: "돌발 이벤트에서 '판단 안 함'은 최악의 선택입니다"

시나리오: "AI 반도체 서프라이즈"
  Day 23: 엔비디아 +15% 급등 (뉴스: "예상 초과 실적")
  → 30초 타이머 시작
  → 추격 매수? 관망? 보유분 익절?
  → 유사 AI: "30% 익절, 70% 보유" (이유: 추가 상승 여력)
  → 최고 AI: "전량 보유" (이유: 3파 상승 초기)
  → 나: 전량 익절 (수익 확정)
  → 피드백: "익절은 좋지만, 최고 AI는 +23% 더 벌었습니다"
```

**핵심**: 돌발 이벤트는 **미리 여러 번 경험해야** 실전에서 대처할 수 있습니다. 실제 데이터에서 발생했던 급등/급락을 반복 훈련하면, 실전에서 같은 상황이 왔을 때 **얼어붙지 않고 판단**할 수 있습니다.

### ★-3. AI/로봇주 실제 1년 데이터 — 가짜가 아닌 진짜 파도

```
[왜 AI/로봇주인가]

일반 주식의 변동성:  ±1~3% (잔잔한 호수)
AI/로봇주의 변동성: ±7~15% (거친 파도)

엔비디아: 2024.11 $140 → 2025.03 $89 (-36%) → 2025.11 $195 (+119%)
에코프로: 2024.11 ₩45,000 → 2025.02 ₩28,000 (-38%) → 2025.08 ₩62,000 (+121%)

→ 이 "진짜 파도"를 타는 연습을 합니다.
→ 가짜 데이터로는 절대 느낄 수 없는 변동성의 공포와 쾌감.
```

**사용 데이터**: 2024.11 ~ 2025.11, 실제 AI/로봇 관련 종목의 일봉 데이터
- 엔비디아, AMD, TSMC (해외 AI 반도체)
- 에코프로, 삼성전자, SK하이닉스 (국내 AI/반도체)
- 테슬라, 현대차 (로봇/자율주행)
- 카카오, 네이버 (AI 플랫폼)

**핵심**: 랜덤 생성 데이터가 아니라 **실제로 일어났던 시장의 파도**입니다. "2025년 3월 AI 버블 붕괴"를 직접 경험하고, "2025년 8월 반등"을 직접 타봅니다. 이 경험이 실전에서 "아, 이 패턴 전에 본 적 있어"라는 감각을 만듭니다.

### ★-4. 나와 비슷한 AI와 같은 주식으로 대결 — 거울을 보는 훈련

```
[왜 "유사 AI"가 핵심인가]

최고 AI만 있으면:
  "최고 AI +25%, 나 +8%"
  → "그래서 뭐, 나는 초보니까..."
  → 학습 동기 없음

유사 AI가 있으면:
  "유사 AI +11%, 나 +8%"
  → "나랑 비슷한 성향인데 3%p 더 벌었네?"
  → "뭐가 달랐지?" → 구체적 학습 동기 발생

유사 AI + 최고 AI 둘 다 있으면:
  "최고 AI +25%, 유사 AI +11%, 나 +8%"
  → 단기 목표: 유사 AI 추월 (+3%p만 더)
  → 장기 목표: 최고 AI에 도전 (+17%p)
  → 성장의 이정표가 명확해짐
```

**같은 주식으로 대결하는 이유:**

다른 앱의 AI는 **다른 종목**을 거래합니다. "AI가 삼성전자로 +15% 벌었어요"라고 해봐야, 내가 카카오를 거래했다면 비교 자체가 무의미합니다.

"파도를 타라"는 **내가 카카오를 샀으면, AI도 카카오에서 결정**합니다.
- 같은 가격, 같은 차트, 같은 뉴스
- 다른 것은 오직 **판단**뿐
- 그래서 비교가 **정확하고**, 학습이 **구체적**입니다

### ★-5. 큰 액수로 미리 느끼는 돈의 감각 — 100만원과 5,000만원은 다른 게임

```
[금액별 심리 변화]

100만원으로 연습:
  카카오 10주 매수 (72,000원 × 10 = 72만원)
  -5% 하락 = -36,000원 손실
  → "에이, 3만원이네" → 손절 안 함 → 나쁜 습관 형성

5,000만원으로 연습:
  카카오 500주 매수 (72,000원 × 500 = 3,600만원)
  -5% 하락 = -1,800,000원 손실
  → "180만원...!" → 심장이 뛴다 → 진지하게 판단
  → 손절할까? 버틸까? → 실전과 같은 고민

1억원으로 연습:
  카카오 1,000주 매수 (72,000원 × 1,000 = 7,200만원)
  -5% 하락 = -3,600,000원 손실
  → "360만원이 날아갔다..." → 공포
  → 이 공포를 가상에서 미리 경험하는 것이 핵심
```

**왜 이것이 중요한가:**

실제 투자를 시작하면 대부분 **소액으로 시작**합니다. 소액에서는 손실이 작아서 **나쁜 습관이 형성**됩니다 (손절 안 함, 물타기, 감정적 매매). 나중에 금액이 커지면 그 습관 때문에 **큰 손실**을 봅니다.

- "파도를 타라"는 **처음부터 큰 금액(500만원~1억원)으로 훈련**합니다. 
- 가상이지만, 30초 타이머와 AI 경쟁이 만드는 긴장감 속에서 **"3,600만원이 -5% 빠졌을 때의 감정"**을 미리 경험합니다. 
- 이 경험이 실전에서 **감정에 휘둘리지 않는 판단력**을 만듭니다.

### ★-6. 정리: "파도를 타라"가 만드는 투자자

```
Before (기존 시뮬레이션):
  ❌ 가짜 데이터로 혼자 연습
  ❌ 차트를 봐도 뭘 해야 할지 모름
  ❌ 급등/급락에서 얼어붙음
  ❌ 100만원으로 연습 → 실전에서 큰돈 감각 없음
  ❌ 뭘 잘못했는지 모른 채 다음 거래

After ("파도를 타라"):
  ✅ 실제 AI/로봇주 1년 데이터로 진짜 파도 체험
  ✅ 3단계 훈련으로 차트가 "보이기" 시작
  ✅ 돌발 이벤트를 수십 번 경험 → 실전에서 침착하게 대응
  ✅ 5,000만원~1억원으로 훈련 → 큰돈의 심리적 압박 면역
  ✅ 유사 AI와 같은 종목 비교 → 정확한 피드백으로 성장
```

## 4. 문제 정의: 왜 이 앱이 필요한가

### 4-1. 현재 투자 교육의 5가지 문제

#### 문제 1: "차트를 아무리 공부해도 실전에서 안 보인다"

- 유튜브 차트 강의 조회수 상위 100개의 평균 시청 시간: **8분**
- 8분 동안 "3파 상승 패턴"을 이해했다고 생각하지만, 실전 차트 앞에서는 **아무것도 보이지 않음**
- **원인**: 차트 감각은 지식이 아니라 **반복 체험**으로만 형성됨
- **"파도를 타라"**: 패턴(15초) → 시나리오(30초) →   시뮬레이션(100턴) 반복 훈련 → 차트가 "느껴지는" 수준까지

#### 문제 2: "급등/급락에서 얼어붙어서 아무것도 못 한다"

- 개인 투자자의 최대 손실은 **돌발 이벤트 당일** 발생 (증권사 데이터)
- 원인: 급등/급락을 **처음 경험**하기 때문 → 패닉 → 최악의 판단 (또는 판단 불가)
- 기존 앱: 돌발 이벤트 시나리오 없음, 시간 압박 없음
- **"파도를 타라"**: 실제 데이터 기반 급등/급락을 **수십 번 반복 경험** → 실전에서 침착하게 대응

#### 문제 3: "혼자 연습하면 뭘 잘못했는지 모른다"

```
기존 앱:
  카카오 500주 매수 → 3일 후 -3.2% → "왜?" → 모름 → 다음 거래

"파도를 타라":
  카카오 500주 매수 (5,000만원 중 3,600만원 투입)
  → 즉시: "유사 AI는 250주만 매수 (분할 매수), 최고 AI는 700주 매수 (공격적)"
  → 3일 후: "최고 AI +8.2%, 유사 AI +5.1%, 나 +3.8%"
  → 피드백: "투자 비중을 높이되, 분할 매수했다면 유사 AI 수준 가능"
```

#### 문제 4: "소액 연습은 나쁜 습관을 만든다"

- 100만원으로 연습 → -5% = 5만원 손실 → "별거 아니네" → **손절 안 하는 습관 형성**
- 나중에 5,000만원 투자 → -5% = 250만원 손실 → 패닉 → 물타기 → 더 큰 손실
- **"파도를 타라"**: 처음부터 **500만원~1억원** 단위로 훈련 → 큰 금액의 심리적 무게를 미리 체득

#### 문제 5: "가짜 데이터로는 진짜 감각이 안 생긴다"

- 대부분의 시뮬레이션 앱: 랜덤 생성 데이터 또는 단순 패턴
- 실제 시장: 뉴스, 실적, 지정학 리스크가 만드는 **예측 불가능한 변동**
- **"파도를 타라"**: AI/로봇주의 **실제 지난 1년 데이터** → "2025년 3월 AI 버블 붕괴"를 직접 경험

### 4-2. 시장 검증: 고객이 원하는 것

| 고객 니즈 | 기존 앱 충족도 | "파도를 타라" 충족도 |
|:---|:---:|:---:|

| 맞춤형 피드백 | ❌ 없음 | ✅ 성향 매칭 AI |

## 5. 솔루션: "파도를 타라"의 핵심 가치

### 5-1. 핵심 기능 4가지

#### 기능 1: 차트 감각 훈련 시스템 — 시나리오 → 패턴 → 시뮬레이션

차트를 "읽는" 것이 아니라 "느끼는" 수준까지 끌어올리는 3단계 훈련:

| 단계 | 방식 | 타이머 | 훈련 목표 |
|:---:|:---|:---:|:---|
| **시나리오** | 실제 사건 기반 5턴 투자 | 30초 | 상황 판단력 — "이 뉴스에 어떻게 반응할까?" |
| **패턴** | 엘리엇 파동 패턴 인식 | 15초 | 반사 신경 — "이 모양은 3파 시작이다" |
| **시뮬레이션** | 100턴 실전 게임 | 30초 | 종합 감각 — 상승/하락/횡보/급등/급락 전부 경험 |

**돌발 이벤트 훈련**: 시뮬레이션 중 실제 데이터에서 발생했던 급등(+10%↑)/급락(-8%↓) 구간을 반드시 통과합니다. 30초 안에 손절/관망/추격매수를 결정해야 합니다. 이 경험을 수십 번 반복하면, 실전에서 같은 상황이 왔을 때 **얼어붙지 않고 판단**할 수 있습니다.

#### 기능 2: 듀얼 AI 비교 — 같은 주식으로 대결 (USP)

**"같은 종목, 다른 판단"** — 이것이 모든 경쟁사와의 결정적 차이입니다.

```
내가 거래한 바로 그 종목에서:

유사 AI (나와 비슷한 성향) — 거울
  → "나와 비슷한데 3%p 더 벌었다. 뭐가 달랐지?"
  → 성향 자동 매칭 (5단계: 보수형 → 초공격형)
  → 단기 목표: 유사 AI 추월

최고 AI (항상 초공격형) — 목표
  → "최고 수준은 이 정도다. 여기까지 가야 한다"
  → 장기 목표: 최고 AI에 도전

같은 주식이기 때문에:
  → 같은 가격, 같은 차트, 같은 뉴스
  → 다른 것은 오직 "판단"뿐
  → 비교가 정확하고, 학습이 구체적
```

#### 기능 3: 실제 AI/로봇주 1년 데이터 — 진짜 파도

랜덤 생성 데이터가 아닌, **실제로 일어났던 시장의 파도**입니다:

| 구간 | 실제 이벤트 | 훈련 효과 |
|:---|:---|:---|
| 2024.11~2025.01 | AI 테마 급등기 (+30~50%) | 상승장에서 탐욕 관리 |
| 2025.02~2025.03 | AI 버블 붕괴 (-25~40%) | 급락장에서 공포 관리 |
| 2025.04~2025.06 | 횡보/회복기 | 인내심, 재진입 타이밍 |
| 2025.07~2025.09 | 로봇주 테마 급등 | 새로운 파도 포착 |
| 2025.10~2025.11 | 실적 시즌 변동 | 이벤트 기반 판단 |

**대상 종목**: 엔비디아, AMD, TSMC, 에코프로, 삼성전자, SK하이닉스, 테슬라, 카카오, 네이버 등

#### 기능 4: 큰 액수 훈련 — 500만원~1억원의 돈 감각

| 초기 자본 | 대상 | 훈련 효과 |
|:---:|:---|:---|
| **500만원** | 투자 입문자 | 첫 투자의 긴장감 체험 |
| **1,000만원** | 소액 투자자 | 분산 투자 감각 |
| **5,000만원** | 중급 투자자 | 포지션 관리, 손절의 무게 |
| **1억원** | 고급 훈련 | 큰돈의 심리적 압박 면역 |

5,000만원 중 3,600만원을 한 종목에 넣었을 때 -5%가 빠지면 **180만원**입니다. 이 숫자를 30초 타이머 속에서 보는 경험 — 이것이 실전 투자 전에 반드시 거쳐야 할 훈련입니다.

## 8. 차별점 & 고객 니즈 매칭

### 8-1. 4가지 결정적 차별점

이 4가지는 **어떤 경쟁사도 제공하지 않는** 우리만의 핵심 가치입니다.

#### 차별점 1: 차트를 읽는 힘 — 3단계 체계적 감각 훈련

```
[기존 앱]
  "자유롭게 매매하세요" → 아무 체계 없이 반복 → 실력 안 늘음

[파도를 타라]
  시나리오(상황 판단) → 패턴(반사 신경) → 시뮬레이션(종합 감각)
  → 체계적으로 차트가 "느껴지는" 수준까지 훈련
```

#### 차별점 2: 돌발 이벤트 대처 — 실제 급등/급락 반복 훈련

```
[기존 앱]
  평탄한 시장만 경험 → 실전 급락에서 패닉 → 큰 손실

[파도를 타라]
  실제 AI/로봇주 데이터의 급등(+15%)/급락(-8%) 구간을 수십 번 통과
  → 30초 안에 판단하는 훈련 반복
  → 실전에서 "아, 이거 전에 해봤어" → 침착하게 대응
```

#### 차별점 3: 같은 주식으로 AI 대결 — 정확한 비교, 구체적 학습

```
[기존 앱의 AI]
  AI가 다른 종목을 거래 → "AI 수익률 +15%입니다"
  → 사용자: "그래서 뭐? 내 종목이랑 다른데?"

[파도를 타라의 AI]
  내가 카카오 500주를 매수할 때 (5,000만원 중 3,600만원 투입):
  → 유사 AI: "카카오 250주 분할 매수" (이유: 지지선 확인)
  → 최고 AI: "카카오 700주 공격 매수" (이유: 모멘텀 돌파)
  → 사용자: "나는 비중도, 타이밍도 중간이었구나"
```

#### 차별점 4: 큰돈 감각 — 500만원~1억원으로 훈련

```
[기존 앱]
  100만원 고정 → -5% = 5만원 → "별거 아니네" → 나쁜 습관

[파도를 타라]
  1억원 선택 가능 → -5% = 500만원 → 심장이 뛴다
  → 가상이지만 30초 타이머 + AI 경쟁 속에서 진짜 긴장감
  → 이 경험이 실전에서 감정에 휘둘리지 않는 판단력을 만듦
```
