### Q1. 손절이 뭔가요? 꼭 해야 하나요?

**손절**은 주가가 내가 산 가격보다 떨어졌을 때, 더 큰 손실을 막기 위해 파는 것입니다.

```mermaid
flowchart LR
    A["100,000원에 매수"] --> B{"주가 하락"}
    B --> C["93,000원 도달"]
    C --> D["손절 실행<br>7,000원 손실 확정"]
    D --> E["✅ 더 큰 손실 방지"]
    
    B --> F["손절 안 함"]
    F --> G["80,000원까지 하락"]
    G --> H["❌ 20,000원 손실<br>회복에 25% 상승 필요"]
```

손절이 필요한 이유:

```
10% 손실 → 회복하려면 11% 올라야 함
50% 손실 → 회복하려면 100% 올라야 함
```

손실이 클수록 회복이 훨씬 어려워집니다. 작은 손실을 인정하는 것이 큰 손실을 막는 방법입니다.

---

### Q2. 익절이 뭔가요? 한 번에 다 팔아야 하나요?

**익절**은 주가가 올랐을 때 수익을 챙기고 파는 것입니다. 한 번에 다 팔 필요는 없습니다.

```mermaid
flowchart TD
    A["매수가: 100,000원"] --> B["+3% → 103,000원"]
    B --> C["20% 팔기<br>소액 수익 확정"]
    A --> D["+5% → 105,000원"]
    D --> E["30% 팔기<br>추가 수익 확정"]
    A --> F["+8% → 108,000원"]
    F --> G["30% 팔기<br>추가 수익 확정"]
    A --> H["트레일링 스탑<br>고점 -3%"]
    H --> I["나머지 20% 자동 청산"]
```

나눠서 팔면 주가가 계속 오를 때도 수익에 참여할 수 있고, 고점을 정확히 맞출 필요도 없습니다.

### Q3. 조건부 주문과 일반 주문의 차이가 뭔가요?

```mermaid
flowchart LR
    A["주문 방식"] --> B["일반 주문"]
    A --> C["조건부 주문"]
    
    B --> B1["내가 직접 버튼 클릭"]
    B --> B2["감정 개입 높음"]
    B --> B3["시장 볼 수 있을 때 유리"]
    
    C --> C1["조건 충족 시 자동 실행"]
    C --> C2["감정 개입 낮음"]
    C --> C3["직장인·학생 등 바쁠 때 유리"]
    
    C --> D["⚠️ 손절은 항상 조건부 주문!"]
```

**어떤 종목이든 손절만큼은 반드시 조건부 주문으로 설정하세요.** 감정이 개입되면 손절 타이밍을 놓칩니다.

### Q4. 트레일링 스탑이 뭔가요?

**트레일링 스탑**은 주가가 오를 때 자동 매도선도 같이 올라가다가, 주가가 꺾이면 자동으로 파는 주문입니다.

```mermaid
flowchart TD
    A["150,000원 매수<br>트레일링 스탑 -10% 설정"] --> B["주가 상승"]
    B --> C["180,000원<br>자동 매도선: 162,000원"]
    C --> D["200,000원<br>자동 매도선: 180,000원"]
    D --> E{"주가 하락?"}
    E -->|"180,000원 이하"| F["🔴 자동 매도 실행!<br>수익 확정"]
    E -->|"계속 상승"| G["자동 매도선도 계속 올라감"]
    G --> E
```

최고점이 어딘지 몰라도 됩니다. 주가가 오르는 동안 계속 보유하다가, 꺾이는 순간 자동으로 팔립니다.

### Q5. 왜 한 번에 다 사거나 팔지 않고 나눠서 해야 하나요? (그라데이션 전략)

```mermaid
flowchart LR
    A["전략 비교"] --> B["흑백 전략<br>한 번에 전량"]
    A --> C["그라데이션 전략<br>단계적으로 조금씩"]
    
    B --> B1["❌ 손절선 도달 → 전량 매도<br>이후 반등하면 억울함"]
    B --> B2["❌ 익절선 도달 → 전량 매도<br>이후 +50% 되면 아쉬움"]
    
    C --> C1["✅ 단계별로 조금씩 팔기<br>반등 시 일부 물량 살아있음"]
    C --> C2["✅ 단계별로 조금씩 팔기<br>계속 오를 때도 수익 참여"]
    C --> C3["✅ 심리적 안정<br>고점 맞출 필요 없음"]
```

파도는 한 번에 꺾이지 않습니다. 조금씩 반응하는 것이 더 유리합니다.

### Q7. 손절을 나눠서 하는 방법은 어떻게 되나요?

```mermaid
flowchart TD
    A["매수가 100,000원"] --> B{"파도 세기?"}
    
    B -->|"🏖️ 대형주"| C["1단계: -2% → 20% 팔기<br>2단계: -3% → 30% 팔기<br>3단계: -5% → 50% 팔기"]
    B -->|"🌊 중형주"| D["1단계: -3% → 25% 팔기<br>2단계: -5% → 35% 팔기<br>3단계: -8% → 40% 팔기"]
    B -->|"🌀 소형주"| E["1단계: -5% → 30% 팔기<br>2단계: -8% → 40% 팔기<br>3단계: -12% → 30% 팔기"]
    B -->|"🌪️ 급등주"| F["1단계: -5% → 50% 팔기<br>2단계: -8% → 50% 팔기"]
    
    C --> G["⚠️ 거래량 증가 확인 시<br>다음 단계 빠르게 실행"]

    E --> G
    F --> G
```

> 손절 신호가 강할수록(거래량 폭발 + 추세 전환) → 더 빠르게 다음 단계를 실행하세요.

### Q11. 이동평균선이 뭔가요? 어떻게 활용하나요?

이동평균선은 최근 N일 동안의 평균 주가를 선으로 이은 것입니다.

```mermaid
flowchart TD
    A["이동평균선 확인"] --> B{"주가와 이동평균선 위치"}
    
    B -->|"주가 > 20일선 > 60일선"| C["📈 오르는 추세<br>보유 유지"]
    B -->|"주가 < 20일선 < 60일선"| D["📉 내리는 추세<br>관망 또는 매도"]
    B -->|"이동평균선 수렴"| E["➡️ 횡보<br>돌파 방향 확인 후 진입"]
    
    E --> F{"돌파 방향?"}
    F -->|"위로 돌파"| G["매수 진입"]
    F -->|"아래로 돌파"| H["관망 유지"]
```

### Q13. 물타기가 뭔가요? 해도 되나요?

**물타기**는 주가가 떨어졌을 때 추가로 더 사서 평균 매수가를 낮추는 것입니다.

```mermaid
flowchart LR
    A["추가 매수"] --> B["❌ 무계획 물타기"]
    A --> C["✅ 계획적 분할 매수"]
    
    B --> B1["즉흥적으로 추가 매수<br>→ 계속 하락 시 손실 폭발"]
    
    C --> C1["-10%: 예산의 30% 매수<br>-20%: 예산의 30% 매수<br>-30%: 예산의 40% 매수"]
    C --> C2["미리 구간과 비율 정해두기<br>→ 감정 개입 없음"]
```

핵심은 **미리 어느 가격에 얼마나 살지 정해두는 것**입니다.

### Q14. 손절 후 바로 다시 사도 되나요?

```mermaid
flowchart TD
    A["손절 실행"] --> B["관망 모드 전환"]
    B --> C{"반등 신호 있는가?"}
    
    C -->|없음| D["계속 관망<br>최소 1~2일"]
    D --> C
    
    C -->|있음| E{"신호 확인"}
    E -->|"거래량 증가 + 추세 전환"| F["재진입 검토<br>→ 1단계 매수만 소량"]
    E -->|"신호 약함"| D
    
    F --> G["재진입 후<br>즉시 손절선 재설정"]
```

손절 후 바로 다시 사면 같은 이유로 또 손절하게 됩니다. **최소 1~2일 관망** 후 반등 신호를 확인하세요.

### Q15. FOMO가 뭔가요? 어떻게 극복하나요?

**FOMO(Fear Of Missing Out)** 는 "나만 수익을 못 챙기는 것 같은 두려움"입니다.

```mermaid
flowchart TD
    A["주가 급등"] --> B["FOMO 발생<br>나도 빨리 사야 해!"]
    B --> C{"체크리스트 확인"}
    
    C -->|"이미 많이 올랐음"| D["❌ 추격 매수 금지<br>다음 파도 대기"]
    C -->|"진입 조건 충족"| E["✅ 계획대로 1단계만 진입"]
    C -->|"조건 불충분"| D
    
    D --> F["💡 다음 파도가 반드시 온다<br>매매 일지로 패턴 파악"]
```

FOMO를 극복하는 방법:
- 매매 전 체크리스트를 반드시 확인하기
- 이미 많이 오른 주식은 추격 매수 금지
- "다음 파도가 반드시 온다"는 생각 갖기
- 매매 일지를 써서 감정적 매매 패턴 파악하기

### Q31. 처분효과가 뭔가요?

**처분효과(Disposition Effect)** 는 투자자가 수익 난 주식은 빨리 팔고, 손실 난 주식은 오래 들고 있는 심리적 편향입니다.

```mermaid
flowchart LR
```

### Q32. 왜 손실 난 주식을 손절하지 못하는 건가요?

**전망이론(Prospect Theory)** 으로 설명할 수 있습니다.

```mermaid
flowchart TD
    A["사람의 손익 감각"] --> B["이익 +10만원의 기쁨"]
    A --> C["손실 -10만원의 고통"]
    
    B --> B1["기쁨 크기: 1"]
    C --> C1["고통 크기: 약 2배\n훨씬 더 아프게 느낌"]
    
    C1 --> D["손실을 확정하기 싫어서\n'곧 오르겠지' 버팀"]
    D --> E["❌ 손실이 더 커짐"]
    
    D --> F["해결책: 조건부 손절 주문\n감정 개입 전에 자동 실행"]
```

손실을 인정하는 순간 고통이 확정되기 때문에, 뇌가 그것을 피하려 합니다. 조건부 주문은 이 고통을 느끼기 전에 자동으로 실행해줍니다.

### Q34. 내가 처분효과에 빠졌는지 어떻게 알 수 있나요?

아래 질문에 솔직하게 답해보세요.

```mermaid
flowchart TD
    A["처분효과 자가 진단"] --> B["수익 중일 때"]
    A --> C["손실 중일 때"]
    
    B --> B1{"'빨리 팔고 싶다'\n'수익이 줄어들까봐 불안하다'"}
    B1 -->|"YES"| B2["⚠️ 처분효과 발동!\n계획된 익절 단계 지키기\n트레일링 스탑 설정 확인"]
    B1 -->|"NO"| B3["✅ 계획대로 진행"]
    
    C --> C1{"'곧 오르겠지'\n'손절선을 내리고 싶다'\n'추가 매수하고 싶다'"}
    C1 -->|"YES"| C2["🚨 처분효과 발동!\n즉시 1단계 손절 실행\n손절선 절대 내리지 말 것"]
    C1 -->|"NO"| C3["✅ 계획대로 진행"]
```

**수익 중 처분효과 신호**
- "조금만 더 오르면 팔겠다"고 계속 미루고 있다
- 수익이 줄어들까봐 불안해서 빨리 팔고 싶다
- 익절 계획 없이 감으로 팔려 한다

**손실 중 처분효과 신호**
- "곧 오를 것 같아서" 손절을 미루고 있다
- 손절선을 아래로 내리고 싶다
- "평균단가 낮추자"며 계획 없이 추가 매수하려 한다

### Q27. OCO 주문이 뭔가요?

**OCO(One Cancels the Other)** 는 손절과 익절을 동시에 설정하는 주문입니다.

```mermaid
flowchart TD
    A["100달러 매수"] --> B["OCO 주문 설정"]
    B --> C["손절: 92달러 이하 시 매도"]
    B --> D["익절: 115달러 이상 시 매도"]
```
